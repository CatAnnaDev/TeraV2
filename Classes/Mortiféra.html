Priorités de compétences
Avant de donner des lignes directrices sur les rotations, nous devons parler des priorités en matière de compétences afin que les lignes directrices aient un sens. Voici

la priorité des compétences:
1 Spirale funeste → double cisaille → coup sinistre + reset → frappe déchirante → 1 spirale funeste → frappe pendulaire → torrent de lame → étreinte fulgurante → déferlement macabre

Exemple de rotation complète : 1 spirale funeste → double cisaille → coup sinistre + reset → frappe déchirante → 1 spirale funeste → frappe pendulaire → torrent de lame → double cisaille → coup sinistre + reset → étreinte fulgurante → déferlement macabre → recommencer la rotation le déferlement macabre ne sera pas up à la deuxieme rotation, le remplacer par le fouet d'ombre

petite astuce :
la 2eme spirale funeste n'est pas obligée d'etre placée apres la frappe déchirante, utile pour reprendre l'enchainement si vous avez du l'interrompre
Exemple : 1 spirale → double cisaille → 1 spirale → coup sinistre → frappe déchirante

Pourquoi est-ce la liste des priorités?
Frappe pendulaire → Torrent de lames doit être la plus haute priorité car elles est l'une de vos meilleure application et aussi la plus rapide.

Rotation de L'Aura Ténébreuse (Burst)
Runs normaux: 1 spirale → coup sinistre → double cisaille → coup sinistre → frappe déchante → répetez
